Portfolio
===================

The purpose of this project is to simply try and keep redesigning it over and over every year with some of the better techniques that I learn during the previous year.

## Technologies used

- Templating: `Pug`
- Styling: `SCSS`
- Scripting: `jQuery and vanilla Javascript`
- JS Compiler: `Babel ES6`
- Linting: `ESLint with airbnb style guide`
- Features: `Autoprefixer and minification`
